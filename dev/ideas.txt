- Xs[K!Pattern = NewValue]
  Set's K value new NewValue only when it matched Pattern.
  For example,
     Root = Root[~!-'/' = '/']
  is the same as:
     less Root.~ >< '/': Root = "[Root]/"`
  Can be further shortened to &Root[~!-'/' = '/']
  Another example
     MusicPath.split^'/'[~!'' = 'index.htm'].text^'/'
   Add 'index.htm' file, if there is no file component at the end.
   Multiple N! per case are allowed. I.e
      Xs[N!M!0=1] replaces Nth and Mth elements by 1 if 0
      Xs[N!0=1] = | Ys Xs; Ys.N = Ys.N[N!0=1]

- `Xs[N!V]` should not really signal out of bound error by default
   but should pad enough elements up to N.
   If index is negative, it adds padding elements to the beginning.
   There should be a way to pick the padding element.
     [a b c][-2!0] produces [0 No a b c]
